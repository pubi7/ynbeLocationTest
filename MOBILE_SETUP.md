# Утас дээр суулгах заавар

## Android утас дээр суулгах

### 1. Утасны тохиргоо

#### Developer Options идэвхжүүлэх:
1. **Settings** → **About phone** руу орох
2. **Build number** дээр **7 удаа** дарах (энэ нь Developer Options-ийг идэвхжүүлнэ)
3. Буцаж **Settings** → **Developer options** руу орох

#### USB Debugging идэвхжүүлэх:
1. **Developer options** дээр орох
2. **USB debugging**-ийг **ON** болгох
3. **Install via USB** (эсвэл **USB installation**) идэвхжүүлэх

### 2. Утасыг компьютерт холбох

1. USB кабелаар утасыг компьютерт холбох
2. Утас дээр "USB debugging зөвшөөрөх үү?" гэсэн мэдэгдэл гарвал **Allow** эсвэл **Зөвшөөрөх** дарна
3. "Энэ компьютерт үргэлж зөвшөөрөх" checkbox-ийг сонгох (сонгосон бол дараа нь мэдэгдэл гарахгүй)

### 3. Компьютер дээр командууд ажиллуулах

```bash
# 1. Утас холбогдсон эсэхийг шалгах
flutter devices

# Хэрэв утас харагдвал дараах командыг ажиллуулах:
flutter run

# Эсвэл APK файл үүсгэх:
flutter build apk --release
```

### 4. APK файл суулгах (шууд утас дээр)

Хэрэв APK файл үүсгэсэн бол:

1. `build/app/outputs/flutter-apk/app-release.apk` файлыг утас руу шилжүүлэх
2. Утас дээр файл manager ашиглан APK файлыг олох
3. APK файл дээр дарж суулгах
4. "Unknown sources" зөвшөөрөх (хэрэв анх удаа бол)

## iOS утас дээр суулгах (хэрэв iPhone ашиглаж байгаа бол)

### 1. Xcode шаардлагатай
- Mac компьютер шаардлагатай
- Xcode суулгасан байх ёстой

### 2. Командууд:
```bash
flutter build ios
# Дараа нь Xcode-оор нээж суулгах
```

## Утас дээр хийх тохиргоо

### 1. Интернэт холболт
- WiFi эсвэл Mobile data идэвхтэй байх ёстой

### 2. Байршлын зөвшөөрөл
- Апп нээгдэхэд "Байршлын зөвшөөрөл" асуух болно
- **Allow** эсвэл **Зөвшөөрөх** дарна

### 3. Settings дээр тохиргоо хийх
- Апп дээр нэвтрэх (Settings → Warehouse Web Sync)
- Backend URL оруулах (жишээ: `http://192.168.1.100:3000`)
- Email болон password оруулах
- "Connect" дарж холбогдох

## Асуудал гарвал

### Утас харагдахгүй байвал:
```bash
# ADB шалгах
adb devices

# Хэрэв "unauthorized" гэж харагдвал:
# Утас дээр "USB debugging зөвшөөрөх" мэдэгдэл дээр "Allow" дарна
```

### Апп ажиллахгүй байвал:
1. Утасыг унтрааж асаах
2. Апп-ийг дахин суулгах
3. Байршлын зөвшөөрөл шалгах (Settings → Apps → Aguulga Business → Permissions)

## Хурдан арга (APK файл ашиглах)

1. Компьютер дээр: `flutter build apk --release`
2. APK файлыг утас руу шилжүүлэх (email, cloud storage, эсвэл USB)
3. Утас дээр APK файл дээр дарж суулгах
4. "Unknown sources" зөвшөөрөх

## Анхааруулга

- **Release build** хийхдээ signing key шаардлагатай (Play Store-д байршуулах бол)
- **Debug build** нь хурдан тест хийхэд тохиромжтой
- Утас дээр backend URL нь утасны IP хаяг байх ёстой (localhost биш!)
